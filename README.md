# News_Portal is a SPA (Single Page Application) to display news items

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 1.6.2.

# Displayed news items by categories

* Angular filters are used to categorize news items
* `ng generate filter [filtername]` used to create the filter

# Displayed Most Viewed news items block 

* Angular Pipes are used to display most visited news items
* `ng g(g for generate) pipe [name]` to create pipe'

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
